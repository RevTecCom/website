A 'Novel' Discord Token Stealer Malware - "Crazy Down"

A new piece of Malware is making its way through Discord. It hides itself as a game on the itch.io platform, calling itself "Crazy Down". It seems to be pretty old in the grand scheme of threats, dating back to about a month, but is still actively updated, or the developer was compromised and pushed a bad update. While the former is more likely, the latter is a small possibility.

We are not sure about anything, but we do know this program steals Discord tokens and attempts to make Discord Nitro gift purchases. While assisting an affected individual with removal, we went through all standard operation procedures.

The malware is distributed as a RAR archive, which contains an installer, below are the MD5 hashes for the archive and the installer, in order:

Setup.rar - feacdc9a0f54f87239be1cc06d12a30e
Setup.exe - 4eef5ea6d623a1baa1e882286b1c848c
Although we cannot set up a proper sandbox to explore this piece of software in depth just yet, we have a few notes on this malware, some as a result of assisting the aforementioned affected individual.

This malware is likely to be a completely new concoction, as existing virus scanners seem to miss it. We would call it a worm, but the author/operator clearly logs into accounts manually to choose his targets, either a sign of a focused attack, or more likely a sign that they couldn't be bothered to go through the hoops to set up an auto-messenger - maybe a bit of both.

The installer seems to be a modified NullSoft installer, set up to attempt to run the malware during the installation.

The installer creates a bunch of contents in the installation directory and in other directories. Some of it is standard installer slop, some of it is definitely suspicious. To list them would be exhaustive, check this static analysis report on VirusTotal.

Upon running build.exe, it is certain that the malware delivered its payload. It makes itself quite obvious: at least 18 instances of build.exe were opened at once on the computer of the affected individual.

Note the following suspicious files written to the installation directory in the report:

e9cf96e11f9e6dedd0df1b194f4b1da0\Wallets\Autofills.txt
e9cf96e11f9e6dedd0df1b194f4b1da0\Wallets\Cookies
e9cf96e11f9e6dedd0df1b194f4b1da0\Wallets\Passwords.txt

Quite obvious, isn't it?

What supports our thesis that this is a token stealer is the fact, which we mentioned at the beginning of this text, that the affected individual was charged for 3 Discord Nitro gifts, along with the fact that the legitimate owner of the account used to send the malware to the affected individual later alerted a group of friends to the fact that they had been compromised.

The malware seems to mess with file associations. We direct you to look at the opened and created registry keys in the report.

There is no one inside the RTC with previous experience in threat management. As such, our understanding of this malware is very limited.
How to know if you've been infected + cleanup

THIS IS NOT EXHAUSTIVE. WE ARE STILL UNCERTAIN OF THE NATURE OF THIS MALWARE. IT SEEMS TO REWRITE FILE ASSOCIATIONS - IF YOU'VE BEEN AFFECTED, WE **HIGHLY RECOMMEND** GOING BACK TO A SYSTEM RESTORE POINT IF YOU HAVE ONE, ALONG WITH FOLLOWING THESE STEPS.

A telltale sign of infection is the Discord desktop application restarting itself, especially when you first ran the installer and the supposed "game". Another sign is an abnormal number of tasks being spawned by the "game" (check Task Manager).

If you are infected, follow these steps:

1. Restart the computer in Safe Mode. Press Windows Key + X, click on "Settings", go to "Update & Security" > "Recovery" > "Restart Now" under Advanced Startup. Once you are in a blue menu with the title of "Choose an option", select "Troubleshoot", then "Advanced Options", and press F4 to restart the computer onto Safe Mode.

2. Remove the folder the malware was installed to, along with the installer and the archive. Empty the Recycle Bin.

3. Remove Discord from automatic startup. Open the Start Menu, Click Settings > Apps > Startup, untick "Discord".

4. Restart the computer once again

5. Run a full malware scan with your preferred scanner, we recommend Malwarebytes

DO THE FOLLOWING ON A SECONDARY DEVICE:

6. Change the passwords to all your important accounts

8. Remove payment information from your Discord account

7. If you have any cryptocurrency wallets, transfer important assets to a secondary wallet.
